An interactive block with the Interactivity API for maintaining site navigation.

== Description ==

A WP interactive block to replicate https://www.udemy.com/course/become-a-wordpress-developer-php-javascript/ menus with gravatar for signed in user, login, register, edit profile options

Replace Nav menu with some optional features.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wsb-user-navigation-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
